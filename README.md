# 🏠 House Price Prediction System

## 📌 Project Overview

This is a Machine Learning web application developed using Python, Scikit-learn, and Streamlit. The application predicts the estimated price of a house based on user inputs such as area, bedrooms, bathrooms, stories, and parking spaces.

---

## 🚀 Features

- Predict house prices instantly
- Interactive Streamlit web interface
- Machine Learning model trained using Linear Regression
- Easy-to-use input form
- Real-time prediction

---

## 🛠 Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

## ▶️ Run the Project

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
